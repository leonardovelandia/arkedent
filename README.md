# Arkedent Dental ERP 

Sistema ERP odontológico completo desarrollado en Laravel para la gestión integral de consultorios dentales.

## 🦷 Módulos del sistema

- Dashboard administrativo con métricas en tiempo real
- Gestión de pacientes
- Agenda y citas
- Historia clínica
- Evoluciones
- Valoraciones
- Imágenes clínicas
- Presupuestos
- Abonos y pagos
- Consentimientos
- Módulo de laboratorio
- Inventario
- Proveedores
- Compras
- Reportes
- Usuarios y roles
- Configuración del sistema

## 📊 Funcionalidades principales

- Control de ingresos y cuentas por cobrar
- Seguimiento de citas diarias
- Alertas de laboratorio vencido
- Gestión completa de pacientes
- Accesos rápidos desde dashboard
- Sistema multi-rol (Administrador, Asistente, Doctor)

## ⚙️ Tecnologías

- Laravel
- MySQL
- Bootstrap
- JavaScript

## 🔐 Seguridad

- Manejo de autenticación
- Control de acceso por roles
- Variables de entorno protegidas (.env)
